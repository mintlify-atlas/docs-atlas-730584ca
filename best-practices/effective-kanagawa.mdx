## Shared State Management

###  No Shared State (Best)

```kanagawa
inline uint32 Pure(uint32 a, uint32 b)
{
    return a + b;  // No synchronization needed
}
```

### Single Atomic Block

```kanagawa
inline uint32 Accumulate(uint32 value)
{
    atomic
    {
        static uint32 _sum = 0;
        _sum += value;
        return _sum;
    }
}
```

## Threading Best Practices

- Use `static for` for fixed parallelism
- Use `pipelined_for` for dynamic parallelism
- Minimize atomic blocks
- Keep atomic sections small

## Compiler Settings

```bash
# Fast compilation
kanagawa --optimize=1 --work-list-size=16

# High quality
kanagawa --optimize=2 --work-list-size=20
```
