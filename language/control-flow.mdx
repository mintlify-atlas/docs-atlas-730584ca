## Conditionals

```kanagawa
if (condition)
{
    handle_true();
}
else
{
    handle_false();
}
```

## Switch Statements

```kanagawa
switch (value)
{
case 0: action_zero(); break;
case 1: action_one(); break;
default: action_default();
}
```

## Loops

### static for

```kanagawa
static for (const auto i : 8)
{
    output[i] = input[i] * 2;
}
```

### Sequential Loops

```kanagawa
for (uint32 i = 0; i < count; i++)
{
    process(i);
}
```
