## Function Declaration

```kanagawa
uint32 Add(uint32 a, uint32 b)
{
    return a + b;
}
```

### Inline Functions

```kanagawa
inline uint32 FastAdd(uint32 a, uint32 b)
{
    return a + b;  // No call overhead
}
```

## Templates

```kanagawa
template<typename T>
inline T Max(T a, T b)
{
    return (a > b) ? a : b;
}

auto result = Max(10u32, 20u32);
```

## Function Types

using Callback = (uint32, bool) -> uint8;
