## pipelined_for

Create parallel threads:

```kanagawa
pipelined_for(8, [](uint32 tid)
{
    process_data(tid);
});
```

## async_exec

Spawn independent threads:

```kanagawa
async_exec([]()
{
    background_work();
});
```

## Thread Synchronization

```kanagawa
atomic
{
    static uint32 _shared;
    _shared++;
}
```
